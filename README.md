# AI Flappy Bird
An AI that plays Flappy Bird by using the NEAT python module

# Requirements
* ```numpy```
* ```pygame```
* ```neat-python```
* ```graphviz```
* ```matplotlib```

# Instructions
Run **```flappybird.py```**
